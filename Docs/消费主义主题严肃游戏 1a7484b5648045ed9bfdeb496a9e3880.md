# 什么是消费主义？

## 消费主义的定义

[消费主义 - 维基百科，自由的百科全书 (wikipedia.org)](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9)

**消费主义**（英语：Consumerism）指相信持续及增加[消费](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB)活动有助于经济的[意识形态](https://zh.wikipedia.org/wiki/%E6%84%8F%E8%AD%98%E5%BD%A2%E6%85%8B)。[[1]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-Jenkins2012-1)创造出在生活态度上对[商品](https://zh.wikipedia.org/wiki/%E5%95%86%E5%93%81)的可欲及需求（多消费是好事）让[资本主义](https://zh.wikipedia.org/wiki/%E8%B3%87%E6%9C%AC%E4%B8%BB%E7%BE%A9)可以提高工资及提高消费。[[2]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-O'Hara1999-2)**消费主义**为[发达国家](https://zh.wikipedia.org/wiki/%E5%B7%B2%E9%96%8B%E7%99%BC%E5%9C%8B%E5%AE%B6)的经济引擎，使[现代人](https://zh.wikipedia.org/wiki/%E7%8F%BE%E4%BB%A3%E4%BA%BA)有购买与获得[商品](https://zh.wikipedia.org/wiki/%E5%95%86%E5%93%81)的社会及经济上的信念及集体情绪。[[3]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-Vineyard2006-3)然而在社会科学领域中，不同领域因各别知识传统对消费主义有不同的定义与诠释[[4]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-ClarkeDoel2003-4)，此差异也反映在如[绿色消费主义](https://zh.wikipedia.org/w/index.php?title=%E7%B6%A0%E8%89%B2%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9&action=edit&redlink=1)[[5]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-Mansvelt2010-5)、[消费者运动](https://zh.wikipedia.org/w/index.php?title=%E6%B6%88%E8%B2%BB%E8%80%85%E9%81%8B%E5%8B%95&action=edit&redlink=1)[[6]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-Kaptan2003-6)、[消费者保护](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B4%B9%E8%80%85%E4%BF%9D%E6%8A%A4)[[7]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-Goldring1998-7)、[消费者权利](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E8%80%85%E6%AC%8A%E5%88%A9)[[8]](https://zh.wikipedia.org/wiki/%E6%B6%88%E8%B2%BB%E4%B8%BB%E7%BE%A9#cite_note-Viswanathan2008-8)等概念及实践上。

## 消费主义的具体表现

[消费主义是如何通过伪造文化来欺骗年轻人的？ - 知乎 (zhihu.com)](https://www.zhihu.com/question/342318692/answer/811637376)

案例一的惯用伎俩是过分强调商品非必要的附加功能，让你忽视商品原生的主要功能。
案例二的惯用伎俩是将商品符号化，并将这种符号延伸为身份地位甚至能力的象征，让你忘了自己到底有几斤几两。
案例三的惯用伎俩是鼓吹商品的万能，让你忘了即使是万金油也无法预防新型冠状病毒感染。

A balanced and solid framework which should be simple enough but also flexible.（没有随机内容的堆砌也要成为一个有一定策略性的游戏），策略和运气怎样取舍？运气成分占比比较大。

没有说教，只是单纯的展示消费主义文化的典型表现并加以夸大乃至荒唐（这种荒唐在游戏中被调整成了一种可以打败对手的策略流派以追求游戏平衡，但正常人应当看到其夸大的成分。）
